Requires Unity 4.5.4 or higher.

A classic styled low polygon moon rocketship ideal to start building a Lander Luner game.

Animated Rudders

3 Levels of detail each with its own diffuse texture with specular channel and normal Map at 2048*2048.

LOD0 - Tri Count = 6036 / Vert Count = 3040

LOD1 -	Tri Count = 2536 / Vert Count = 1290

LOD2 -	Tri Count = 1658 / Vert Count = 851


Web Demo: http://www.ajportfolio.co.uk/demo/moonlanderonerocketship.html

Youtube Video: https://www.youtube.com/watch?v=gC6DidmLmbI


Includes 2 bonus low polygon asteroids with diffuse texture with specular channel and normal Map at 2048*2048. Single Level of Detail.
